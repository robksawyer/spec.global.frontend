General info

This page demonstrates simple data fetch from backend via WebSockets and how to show that data in simple list. Within this example application fetches publishers data with related author data populated to each one. Example also covers data pagination, sort and search functions. Actual data is fetched from following backend endpoint. 

GET {{backendConfig.url}}/publisher

Note that all data communications to backend requires JSON Web Token JWT. This JWT is used to authenticate current request to make sure that user is allowed to access that data.

Functions in this example

Data fetch
From backend via WebSockets using $sailsSocket service.
Search
Specify which columns are used in search and type your search words to see results while you're typing.
Pagination
By default, the page displays an example of ten publishers per page. And you can change the number to be displayed per page.
Sorting data
Simple by clicking of column header. This will fetch data from backend again. Unfortunately not from relation columns... yet.
Live updates
Data is updated automatic to GUI if someone changes data that is shown currently.
Navigation
Access to single publisher and author pages simply by click publisher or author name.
